## Section 1: Project Title and Description
GitHub Peru Analytics is a data analytics platform that extracts, processes, and visualizes information about the Peruvian developer ecosystem using the GitHub API. This project connects to GitHub, collects 1,000+ repositories associated with Peru, classifies them using GPT-4 into official CIIU industry categories, calculates relevant developer metrics, and presents them in a Streamlit dashboard.
